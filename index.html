Plant Generator

This a project idea i have in mind.

I'd like to make a plant generator in Unity3D. I loved the graphics of Besiege so i'd like to imitate it using this kind of global illumination and ambient occlusion you can see in the game.

I thought about using a L-system for the generation of the plant because it's one of the first thing that comes in mind when you want to generate branches and leaves but i don't know if i'll stick to it. I'd like this generator to be as customizable as possible, so you could make various kind of plants going from a cactus to a lovely rosebush.
